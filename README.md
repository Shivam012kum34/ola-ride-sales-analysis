# Ola Ride Sales Analysis (EDA)

## Overview
This project performs Exploratory Data Analysis (EDA) on ride booking data to understand booking trends, missing values, correlations, and operational patterns.

## Dataset
- File: `Bookings-100000-Rows.xlsx`
- Records: ~100,000
- Purpose: Academic / learning-based analysis

## Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib

## Project Structure
- `eda_ola_bookings.py` → Main EDA script
- `eda_outputs/` → Generated CSV summaries & plots
- `requirements.txt` → Python dependencies
- `Bookings-100000-Rows.xlsx` → Dataset

## Key Analysis
- Missing value analysis
- Numerical & categorical summaries
- Correlation matrix & heatmap
- Data cleaning & preprocessing

## How to Run
```bash
pip install -r requirements.txt
python eda_ola_bookings.py
