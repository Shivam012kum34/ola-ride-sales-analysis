# Ola Ride Sales Analysis

## Overview
Exploratory data analysis on ride-level data to understand demand patterns,
revenue behavior, and peak-hour inefficiencies.

## Tools Used
- Python (Pandas, NumPy)
- SQL
- Power BI

## Key Analysis
- Demand vs time
- Revenue distribution
- Peak hour trends
- Customer behavior patterns

## Outcome
Generated actionable insights to support pricing optimization
and operational efficiency.
