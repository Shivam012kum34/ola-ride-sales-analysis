# Ola Ride Sales Analysis (EDA)

## Overview
This project performs exploratory data analysis on ride booking data to understand
booking trends, missing values, correlations, and operational patterns.

## Tools & Libraries
- Python
- Pandas
- NumPy
- Matplotlib

## Key Analysis
- Missing value analysis
- Numerical & categorical summaries
- Correlation matrix & heatmap
- Data cleaning and preprocessing

## Outputs
All generated outputs are saved in the `eda_outputs/` folder:
- CSV summaries
- Correlation matrix
- Heatmap visualizations

## Notes
Raw data is not included to keep the repository lightweight and professional.
